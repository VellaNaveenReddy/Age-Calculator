# 🎂 Simple Age Calculator Website

This repository contains a simple **Age Calculator web application** built using **HTML, CSS, and JavaScript**.  
The website allows users to calculate their exact age based on their birthdate.

---

## 🚀 Features

- ✅ User-friendly interface  
- 📅 Input field for selecting birthdate  
- 🎯 One-click age calculation  
- 📊 Displays age in years, months, and days  
- ⚡ Instant result output  

---

## 🛠️ Technologies Used

- 🌐 HTML – Structure of the website  
- 🎨 CSS – Styling and layout  
- ⚙️ JavaScript – Age calculation logic

## ▶️ How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/age-calculator.git
   ```

2. Open the project folder.

3. Run `index.html` in any web browser.

4. Enter your birthdate in the format:
   ```
   YYYY-MM-DD
   ```

5. Click on the **Calculate** button.

6. Your exact age will be displayed instantly.

---

## 📸 Output Example

- Input: 2003-05-10  
- Output: You are 22 years, 1 month, 18 days old (example)
